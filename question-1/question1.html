Question 1 - HTML Terminology

  1. HTML: Hyper Text Markup Language. CSS: Cascading Style Sheets
  2. HTML
  3. CSS
  4. HTML Tag: A set of characters constituting a formatted command for a web page and used to hold the HTML element
  5. Start Tag, Elemental Content and End Tag
  6. The doctype declaration represents the document type, and helps browsers to display web pages correctly.
  7. HTML element: A component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document.
  8. The head element contains meta information about the HTML page.
  9. The body element defines the document's body, and is a container for all the visible contents
  10. Use the <p> tag
  11. Choose between <h1> - <h6> tags
  12. There are 6 different headings. The difference between them are the size of the fonts and defines which headings are more important (h1) and least important (h6)
  13. You can use the <b> tag or the <strong> tab
  14. Use the <i> tag
  15. A nested element (child element), is inside another html element (parent element) and we create a parent and child relationship between them.
  16. Children/Siblings relationship. Inline Elements.

  18. <ul>
